# Офис: доступ, гости, парковка

## Пропускной режим

Офис на Заречной набережной, 12 работает для сотрудников круглосуточно
по бесконтактному пропуску. Утерю пропуска сообщайте на ресепшен —
старый блокируется сразу, новый выдаётся в течение рабочего дня
(первый перевыпуск бесплатно, повторные — 500 ₽ из зарплаты).

## Гости

Гостевой пропуск заказывается в «Астра-Офис» минимум за 2 часа до визита:
ФИО гостя и время. Гость перемещается по офису только в сопровождении
принимающего сотрудника; переговорные первого этажа доступны гостям
без ограничений.

Кандидаты на собеседования оформляются рекрутером автоматически при
назначении встречи.

## Парковка

Подземный паркинг — 40 мест, бронирование через «Астра-Офис» с 18:00
предыдущего дня, максимум 2 брони в неделю на сотрудника. 6 мест
с зарядками для электромобилей бронируются отдельной категорией.
Гостевая парковка (4 места) заказывается вместе с гостевым пропуском.

## Переговорные

Переговорные бронируются из календаря; встреча без брони уступает место
встрече с бронью. Комнаты для звонков (телефонные будки) не бронируются —
живая очередь, лимит 1 час.
