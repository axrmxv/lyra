# Подключение к корпоративному VPN

## Когда нужен VPN

VPN обязателен для доступа к внутренним системам («Астра-HR», Git,
тестовые стенды, базы знаний) из любой сети вне офиса, включая домашнюю.

## Установка клиента

1. Скачайте клиент **AstraConnect** из внутреннего портала
   `https://portal.astra-link.ru/vpn` (доступен без VPN).
2. Установите и запустите; сервер подключения — `vpn.astra-link.ru`,
   порт по умолчанию 443.
3. Войдите с корпоративной учётной записью. Подтвердите вход кодом из
   приложения-аутентификатора (MFA обязателен).

## Типовые проблемы

- **Ошибка «Server unreachable»** — проверьте, не блокирует ли ваш
  провайдер порт 443 для UDP; в настройках клиента переключите протокол
  на TCP.
- **Постоянные разрывы** — отключите энергосбережение сетевого адаптера;
  при работе через мобильный интернет включите режим «Экономия трафика»
  в AstraConnect.
- **Код MFA не подходит** — проверьте синхронизацию времени на телефоне.

Если проблема не решается — заявка в техподдержку с приложением лога
клиента (меню «Помощь» → «Экспорт лога»).

## Ограничения

Одновременно допускается не более **2 активных VPN-сессий** на сотрудника.
Сессия автоматически завершается после 12 часов непрерывного подключения.
